## 📌 Project Overview

A smart hospital queue management app prototype designed to transform the hospital waiting experience from stressful and uncertain into calm, engaging, and informative.

Built using the **Design Thinking process** (Empathize → Define → Ideate → Prototype → Test), the app addresses real patient pain points through empathy-driven features.

> **Problem Statement:** *"How might we transform hospital waiting from a stressful, uncertain experience into a calm, engaging, and informative journey that makes patients feel supported and valued?"*

---

## 🧠 Design Thinking Process

### Stage 1 — Empathize & Define
- Identified 6 key patient problems in hospital waiting areas
- Created 8 detailed user personas
- Built empathy maps covering what patients think, feel, see, hear, say, and do
- Defined a clear, user-centered problem statement

### Stage 2 — Ideate
- Brainstormed solutions around 5 compulsory features
- Prioritized features based on user impact and feasibility
- Focused on inclusivity (elderly, children, hearing/vision-impaired users)

### Stage 3 — Prototype
- Designed 5 interactive app screens in Figma
- Clean, modern UI with indigo/blue theme
- Bottom navigation bar for easy screen switching

### Stage 4 — Test
- Tested with 5 users across different age groups
- Collected feedback on usability, readability, and engagement
- Iterated: increased font sizes, improved button contrast, added more content

## ✨ Key Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | **Queue Progress Tracker** | Real-time position, progress bar, wait time & estimated appointment time |
| 2 | **Distraction Hub** | Relaxing videos, mini games, audio content & podcasts |
| 3 | **Silent Notifications** | Gentle vibration alerts when turn is approaching |
| 4 | **Mood-Based Calming Support** | Breathing exercises, calm sounds, meditation, affirmations |
| 5 | **Feedback System** | Star ratings, open text, checkbox areas, anonymous submission |
